# 🇮🇳 BharatVidhi — Your Guide to the Indian Constitution

> *Explore the core principles of Justice, Liberty, and Equality enshrined in the Indian Constitution.*

---

## 📖 About the Project

**BharatVidhi** is a constitutional education web platform that makes the Indian Constitution accessible, explorable, and easy to understand for every citizen. Built as a DBMS academic project at Christ College of Engineering, it combines a clean heritage-themed UI with a Node.js backend to deliver constitutional articles, principles, and knowledge in an organized and user-friendly way.

Whether you're a student, a legal enthusiast, or just a curious citizen — BharatVidhi is your digital guide to understanding the foundation of Indian democracy. 🏛️

---

## ✨ Features

- 🔐 **User Authentication** — Secure login and signup system (Node.js + Express backend)
- 📜 **Explore the Constitution** — Browse articles, fundamental rights, and constitutional basics
- 📚 **Courses & Tutorials** — Learn about the Constitution through structured content
- 📖 **Resources Section** — Reference materials and additional reading
- 💎 **Premium Access** — Get Premium page for extended content
- 📬 **Contact Page** — Reach out and connect
- ⚙️ **Settings & Profile** — User dashboard and profile management
- 🎨 **Heritage-themed UI** — Elegant design inspired by the original Indian Constitution manuscript

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your system
- npm (comes with Node.js)

### Installation & Running

```bash
# Step 1 — Clone the repository
git clone https://github.com/grazpaulx/Bharathvidhi.git

# Step 2 — Navigate into the backend folder
cd Bharathvidhi/backend

# Step 3 — Install dependencies
npm install

# Step 4 — Start the server
node server.js
```

### Step 5 — Open the app
Open your browser and go to:
```
http://localhost:3000
```

Or simply open `frontend/index.html` directly in your browser if you want to run it without the backend. 🌐
